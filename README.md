# FormVu Docker Image

## Build image
```
cd docker
docker build -t idrsolutions/formvu . 
```

## Run container
Full details on how this Docker Image can be set up and used can be [found here](https://support.idrsolutions.com/formvu/tutorials/cloud/docker).
